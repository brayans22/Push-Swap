# ➡︎ Push Swap ⬅︎

### Summary
This project consists of sorting a list of numbers in ascending order using some moves 
(PB, PA, RRA, RRB, SA, SB, RA, RB) and a sorting algorithm (Radix Sort).

### Usage
1) Compile the code
   ```makefile
   make
   ```
2) Execute push_swap
   ```bash
   ./push_swap "numbers"
   ```
